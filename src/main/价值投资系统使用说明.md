# 价值投资系统使用说明

## 概述

本系统基于《价值投资系统建构举隅》文档中的方法，实现了基于预估收益率的自动化价值投资决策系统。

## 核心功能

1. **买入条件判断**：自动评估股票是否符合买入条件
2. **卖出条件判断**：监控持仓股票是否需要卖出
3. **换股条件判断**：评估是否应该换股
4. **组合管理**：自动管理投资组合，控制行业和股票数量
5. **资金再平衡**：定期调整持仓比例

## 系统参数（默认值）

根据文档设定，系统默认参数如下：

- **最小增长率 (g)**: > 10%
- **最大PEG**: < 1.2
- **最小目标收益率**: > 35%
- **卖出收益率阈值**: <= 0%
- **最大股票数量**: 20只
- **最大行业数量**: 10个
- **每行业最大股票数**: 5只
- **评估年数 (n)**: 3年
- **最大可信增长率**: <= 35%

## 买入条件

股票需同时满足以下所有条件才能买入：

1. PEG < 1.2
2. 目标预估收益率 > 35%
3. 当前市盈率 <= 历史均值市盈率
4. 增长率 (g) > 10%
5. 组合股票数量 < 20只
6. 该行业股票数量 < 5只

## 卖出条件

满足以下任一条件即卖出：

- **A. 基本面变坏**: 增长率 (g) < 10%
- **B. 高估**: 卖出预估收益率 <= 0%

## 换股条件

换股需同时满足：

1. 换入标的与换出标的的目标收益率差 > 35%
2. 换入标的满足所有买入条件
3. 换股后仍满足组合限制

## 使用方法

### 方法1：使用命令行脚本（推荐）

```bash
# 使用默认股票列表（zz500_stocks.csv）
python src/main/run_value_investment.py

# 指定股票列表文件
python src/main/run_value_investment.py --stock-list data/self_selected_stocks.csv

# 指定输出目录
python src/main/run_value_investment.py --output-dir data/my_results

# 指定目标日期
python src/main/run_value_investment.py --target-date 2024-12-31
```

### 方法2：在Python代码中使用

```python
from value_investment_system import ValueInvestmentSystem

# 创建系统实例
system = ValueInvestmentSystem()

# 评估股票列表
stock_codes = ['000001', '600000', '600036']
results_df = system.evaluate_stock_list(stock_codes)

# 选出最佳股票
best_stocks = system.select_best_stocks(stock_codes)

# 查看组合摘要
summary = system.get_portfolio_summary()
print(summary)
```

### 方法3：自定义配置

```python
from value_investment_system import ValueInvestmentSystem

# 自定义配置
custom_config = {
    'min_target_return': 0.30,  # 降低目标收益率要求到30%
    'max_stocks': 15,  # 减少最大股票数量到15只
    'max_stocks_per_industry': 3,  # 每行业最多3只
}

system = ValueInvestmentSystem(config=custom_config)
```

## 输出文件说明

运行脚本后会生成以下文件：

1. **value_investment_all_[timestamp].xlsx**: 所有股票的评估结果
2. **value_investment_buyable_[timestamp].xlsx**: 符合买入条件的股票
3. **value_investment_portfolio_[timestamp].xlsx**: 最终选出的投资组合

## 计算公式

### PEG计算
```
PEG = PE / (g * 100)
```

### 目标预估收益率计算
```
目标收益率 = ((PE_mean / PE_now)^(1/n)) * (1 + g) - 1
```
其中：
- PE_mean: 历史平均市盈率
- PE_now: 当前市盈率
- g: 增长率（小数形式）
- n: 年数（默认3年）

## 注意事项

1. **数据依赖**：系统依赖akshare和baostock获取股票数据，需要网络连接
2. **数据质量**：部分股票可能缺少券商研究报告，无法获取增长率预测
3. **参数调整**：建议根据市场情况和个人风险偏好调整系统参数
4. **定期更新**：建议定期（如每月）重新评估组合，检查卖出条件
5. **再平衡**：系统支持定期再平衡功能，建议每半年或一年执行一次

## 常见问题

**Q: 为什么有些股票评估失败？**
A: 可能原因：1) 股票代码错误 2) 缺少历史PE数据 3) 缺少券商研究报告

**Q: 如何调整系统参数？**
A: 创建ValueInvestmentSystem实例时传入config字典，覆盖默认参数

**Q: 系统会自动交易吗？**
A: 不会。系统只提供投资建议，不执行实际交易

**Q: 如何保存和加载组合？**
A: 使用`system.save_portfolio(file_path)`保存，`system.load_portfolio(file_path)`加载
